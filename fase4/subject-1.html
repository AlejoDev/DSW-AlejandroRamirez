Etiquetas semánticas, Listas y enlaces

Este sitio se desarrollará un Objeto virtual de aprendizaje, con el fin de dinamizar la estrategia de aprendizaje para el diseño de sitios web, generando una fuente de fácil entendimiento y comprensión para los estudiantes, apoyándose en recursos multimedia.

Antes que nada, debemos tener en cuenta que HTML se rige por las etiquetas. Éstas definen cómo es interpretado por el navegador el archivo .html creado con las mismas.

Resumiremos las etiquetas más importantes y su uso.

Destacaremos las principales, comenzando por las etiquetas de texto.

  • Texto
    • <hx>: Es la etiqueta usada para títulos o subtítulos donde x, representa un número del 1 al 6, en donde el 1 es el tamaño más grande que puede tomar con la etiqueta h (<h1>,<h2>,<h3>,<h4>, <h5>,<h6>).

      Ejemplo de la etiqueta <h4>

    • <p>: Etiqueta usada para representar un párrafo, en donde por defecto, los navegadores asignan un margen en la parte superior.
    • <pre>: Representa un texto con formato predefinido,, como si fuera un código de programación. 
      Ejemplo de la etiqueta <pre>
    • <span>: Es una etiqueta que puede contener, desde un caracter hasta una palabra o párrafo, que inicialmente no le aplica ningún estilo a su contenido, pero que con CSS, puede personalizar su estilo.
    • <br>: Se usa para dar saltos de línea.
    • <em>: Etiqueta usada para dar énfasis, muestra el texto en letra cursiva.
      Ejemplo de la etiqueta <em>
    • <strong>: Esta etiqueta se usa para dar importancia al texto, mostrándolo en negrilla.
      Ejemplo de la etiqueta <strong>
    • <u>: Esta etiqueta es usada para que el texto que se encuentre en ella se muestra subrayado
      Ejemplo de la etiqueta <u>
    • <p>:Esta etiqueta es usada para resaltar el texto.
      Ejemplo de la etiqueta <mark>
    • <small>:: Se usa para que las palabras contenidas en la etiqueta, se muestren en un texto más pequeño.
      Ejemplo de la etiqueta <small>
    • <cite>: Es la etiqueta comunmente usada para citar el autor de algo en particular.
      Ramírez, A. (2021). Etiquetas semánticas, Listas y enlaces. Ejemplo de <cite>
    • <a>: Se usa para agregar un enlace al documento. los attributos más comunes usados en esta etiqueta son href y target.
      Ejemplo de la etiqueta <a>
    • <ul>: Se usa para generar listas sin ordenar. Cada uno de los elementos de la lista, se debe generar con las etiquetas.<li>
    • <ol>: Se usa para generar listas con un orden ordenar. Cada uno de los elementos de la lista, se debe generar con las etiquetas.<li>

  • Elementos Estructurales

    Dentro de las etiquetas que definen la estructura tenemos las siguientes:

    • <html>: Con esta etiqueta se da inicio y fin al código html. Puede incluir un atributo llamado lang en el cual se define el idioma del contenido del archivo html.
    • <head>:En esta etiqueta se anida toda la información que se requiere para configurar la página, como los son el título, el tipo de codificación de los carácteres del mismo, así como la ubicación de los archivos externos requeridos, como lo son CSS y Javascript.
    • <body>: Delimita todo el código que genera el contenido del documento, básicamente la parte visible del mismo.
  • Para distribución dentro de la página.
    • <div>: Se usa para realizar una división genérica en el documento.
    • <section>: Este elemento define una sección genérica.
    • <aside>: Este elemento define una división que contiene información relacionada con el contenido principal pero que no es parte del mismo, como referencias a artículos o enlaces que apuntan a publicaciones anteriores.
    • <article>: Se usa para representar un artículo independiente.
    • <header>: Se usa para definir cabeceras.
    • <footer>: Se usa para definir pie de página.

Gauchat, J. (2017). El gran libro de HTML5, CSS3 y JavaScript. Tercera Edición.
