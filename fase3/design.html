¿Qué encontrás acá?

Este sitio se ha desarrollado con el fin de practicar el desarrollo de sitos web, implementando el manejo básico de HTML y de CSS, implementando en su desarrollo el uso de repositorios (Git).

Cuenta con un Layout inicial de un nav, el cual cuenta con un tétulo, un menú de navegació, un container que alberga dos columas, una con una imagen y otra con un texto explicativo.

Con el desarrollo de esta actividad, se logra evidenciar que CSS es el que realza el contenido de un sitio web y que permite manipular el mismo, adaptándolo a los requerimientos y dispositivos desde donde sea ejecutada esta página web.
